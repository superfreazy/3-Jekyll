地址：[https://superfreazy.github.io](https://superfreazy.github.io)

### 安装说明

1. fork库到自己的github
2. 修改名字为：`username.github.io`
3. clone库到本地，参考`_posts`中的目录结构自己创建适合自己的文章目录结构
4. 修改`_config.yml`配置项
5. All done!
